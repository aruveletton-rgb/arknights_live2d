# Placeholder Operator

This directory is reserved for the default Live2D character.

Put a Cubism `*.model3.json` file and its related `moc3`, texture, motion, and expression assets here, then set the client model path to:

```text
/characters/placeholder_operator/your_model.model3.json
```

The current client displays a built-in placeholder operator when this model file is absent, so Mock mode can run without external assets.
